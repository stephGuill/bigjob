 DONNÉES
----------

Les données sont stockées en LocalStorage du navigateur :
  • users        → Utilisateurs et leurs rôles
  • reservations → Demandes de présence

Session utilisateur en SessionStorage (déconnexion à la fermeture)


 RÈGLES IMPORTANTES
----------------------

1. Seuls les emails @laplateforme.io sont acceptés
2. Les dates passées ne peuvent pas être modifiées
3. Une seule demande par date et par utilisateur
4. Les demandes ne peuvent être annulées qu'avant la date


 DÉPANNAGE
------------

Problème : La page ne s'affiche pas
→ Vérifiez que WAMP est démarré
→ Vérifiez l'URL : http://localhost/bigjob/

Problème : Les données ne se sauvegardent pas
→ Vérifiez que LocalStorage est activé dans votre navigateur
→ Essayez de vider le cache : Ctrl+Shift+Delete

Problème : Impossible de se connecter
→ Utilisez les comptes de test fournis
→ Vérifiez que le mot de passe est correct

Problème : Le calendrier ne s'affiche pas
→ Vérifiez la console JavaScript (F12)
→ Assurez-vous que jQuery et Bootstrap sont chargés
